# 🎓 Student Result Manager

A beginner-friendly **Python console application** to manage student marks and results.
This project allows users to add students, view records, check pass/fail results, and delete student data.

---

## 🚀 Features

* Add new student records
* View all students and marks
* Check individual result status
* Delete student records
* Menu-driven console interface
* Beginner-friendly Python project

---

## 🛠️ Technologies Used

* Python 3
* Dictionary Data Structure
* Loops
* Conditional Statements
* Functions (optional upgrade)

## ▶️ How to Run

1. Install Python 3 on your system
2. Clone this repository

```bash
git clone https://github.com/your-username/student-result-manager.git
```

3. Go to project folder

```bash
cd student-result-manager
```

4. Run the program

```bash
python main.py
```

* Marks **Below 40** → Fail ❌
